# WebUI Examples - Go

### Windows
```sh
git clone https://github.com/alifcommunity/webui.git
cd webui\examples\Go\hello_world
go clean
go build -o hello_world.exe
hello_world
```

### Linux
```sh
git clone https://github.com/alifcommunity/webui.git
cd webui/examples/Go/hello_world
go clean
go build -o hello_world
./hello_world
```

### macOS
